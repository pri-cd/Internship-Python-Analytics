## 🛠️ Installation & Usage

1. **Clone the repo**:
    ```bash
    git clone https://github.com/pri-cd/Internship-Python-Analytics.git
    ```

2. **Install dependencies**:
    ```bash
    pip install -r requirements.txt
    ```

3. **Run the notebook**:
    Open the desired `.ipynb` file in Jupyter Notebook.
